===  Mustache template engine for PL/pgSQL

==== Features

- [x] variable unescaped `{{&name}}` and `{{{name}}}`
- [x] variable escaped `{{name}}`
- [ ] conditions
- [ ] loop
- [ ] contexts
- [ ] sections
    * [ ] list
    * [ ] non-false values
    * [ ] lambdas
    * [ ] inverted
- [ ] comments
- [ ] partials
